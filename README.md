# GearForBikes

## Описание проекта 

GearForBikes — это учебный проект интернет-магазина на Django, специализирующегося на продаже городских велосипедов, фиксов, синглспидов, грейвелов и качественных комплектующих.

Ассортимент магазина включает: 

- Синглспиды и фиксированные велосипеды (Fixe Gear)
- Грейвелы, туринги и стальные дорожные комьютеры
- Велопокрышки, камеры и элементы трансмиссии (цепи, коги, звезды)
- Педали, трековые стрепы, спортивные седла и обмотки руля
- Велосумки, аксессуары для байкпакинга и фляги
- Велосипедные инструменты, насосы и парафиновые смазки

## Установка и запуск

### 1. Клонироование репозитория
```bash
git clone  <ссылка_на _репозиторий> 
```

### 2. Создание и активация виртуального окружения
```bash
python -m venv .venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt 
```

### 4. Выполнить миграции 
```bash
python manage.py migrate
```

### 5. Запуск сервера 
```bash
python manage.py runserver
```
После запуска сайт бует доступен http://127.0.0.1:8000/
